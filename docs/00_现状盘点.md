# 00 现状盘点

## 1. 当前项目技术栈

- React
- TypeScript
- Vite
- CSS Modules
- SVG/CSS animation
- GSAP、Framer Motion 已安装但当前核心页面未依赖它们

## 2. 当前已有页面和组件

- `src/app/App.tsx`：当前入口。
- `src/features/course/`：课程外壳、课程时间线、模块视图、教师讲解面板、课程进度。
- `src/features/presentation/`：Presentation Core，负责阶段、播放、暂停、重置。
- `src/features/disaster-demo/landslide/`：滑坡动态演示。
- `src/data/courseModules.ts`：当前课程模块数据。

## 3. 已经实现的课程模块

当前已有 10 个模块，但结构偏上一版：

1. 课程导入
2. 什么是地质灾害
3. 常见六类地质灾害
4. 四川重点防范三大灾害
5. 滑坡专题
6. 崩塌专题
7. 泥石流专题
8. 前兆信号识别
9. 避险互动测试
10. 总结口诀

## 4. 可以保留的内容

- React/Vite/TypeScript 工程基础。
- `PresentationController` 的阶段、播放、暂停、重置能力。
- `LandslideDemo` 和滑坡分层视觉结构。
- 课程 Shell 的顶部进度、右侧教师讲解、底部课程时间轴。
- 当前深色展厅视觉方向。

## 5. 需要调整的内容

- 课程导入需要缩短成 2-3 分钟的暴雨山区快速导入。
- 课程模块需要改为 v2 指定的 10 个模块。
- 避险测试不能作为主流程重点，应弱化为快速复盘或轻量互动。
- 崩塌、泥石流不能只停留在静态卡片，需要补轻量动态过程演示。
- 定义页、六类总览、形成原因、避险流程需要更强的可视化。

## 6. 需要新增的内容

- 形成原因实验室：降雨量、坡度、人为扰动三个滑杆。
- 四川重点三类灾害入口，带跳转到对应演示模块。
- 崩塌动态演示。
- 泥石流动态演示。
- 科学避险流程 + 前兆分类墙。
- 另外三类拓展 + 总结口诀合并模块。
