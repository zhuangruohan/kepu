# 02 本轮迭代任务

## 目标

把当前网页升级为 v2 动态互动课程页，适合 60 分钟课堂讲解、投屏和录屏。

## 必做

1. 调整课程数据为 v2 10 模块。
2. 缩短首页导入，强化暴雨山区视觉。
3. 增加定义页的“自然现象 vs 地质灾害”判断流程。
4. 增加六类灾害总览动画卡片。
5. 增加形成原因实验室滑杆。
6. 增加四川重点三类灾害入口。
7. 保留并增强滑坡完整演示内容。
8. 新增崩塌、泥石流轻量动态演示。
9. 新增科学避险流程和前兆分类墙。
10. 合并另外三类拓展和总结口诀。

## 不做

- 不重写 `PresentationController`。
- 不引入 Three.js。
- 不做后端。
- 不做登录。
- 不做题库型网页。
- 不扩展文件主线以外的大量专业知识。

## 验收命令

- `npm run build`
- 如本地服务已启动，检查 `http://127.0.0.1:5174`
