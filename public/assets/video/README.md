# 视频素材

本目录用于放置真实案例视频。前端引用路径从 `/assets/video/` 开始，不写 `/public`。

当前接入文件：

- `landslide/landslide-case-01.mp4`
- `collapse/collapse-case-01.mp4`
- `debris-flow/debris-flow-case-01.mp4`
- `safety/safety-warning-signals-01.mp4`

要求：

- 默认静音，允许用户点击播放。
- 不自动全屏播放。
- 不遮挡核心演示区域、知识卡、前兆分类墙或互动题。
- 若视频来自官方或公开平台，讲解稿和页面备注需标明来源。
